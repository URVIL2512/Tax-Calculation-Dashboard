Tax Calculation Dashboard
A modern web application built using React and Material-UI that enables users to calculate income tax liabilities under both the Old and New tax regimes in India. This tool is designed for accuracy, simplicity, and efficiency, making it useful for individuals, tax professionals, and financial advisors.

Features
Dual Tax Regime Support
Calculate taxes under both Old and New Indian tax regimes as per current regulations.

Interactive Form Interface
Step-by-step input process with intuitive navigation and real-time validation.

Comprehensive Tax Computation
Calculates income tax based on the latest Indian tax laws, deductions, and exemptions.

Modern UI/UX
Built with Material-UI (MUI) for a sleek, clean, and responsive design.

Real-Time Feedback
Dynamic updates and validation messages for a smooth user experience.

Detailed Tax Reports
Generates complete tax breakdowns and downloadable reports.

Technology Stack
Layer	Tools & Libraries
Frontend	React 18 with TypeScript
UI	Material-UI (v5), Tailwind CSS, Emotion
Styling	Styled Components, Emotion
Charts	Recharts
Reports	jsPDF
Icons	Lucide React Icons
Build Tool	Create React App

Core Functionality
Income Collection
Add all income sources such as salary, HRA, and allowances.

Deduction Management
Apply deductions like 80C, 80D, 80TTA, HRA exemption, and more.

Regime Selection
Compare tax outcomes under both regimes and choose the better one.

Instant Tax Calculation
Tax is computed dynamically based on your inputs.

Report Generation
View and download a detailed tax report in PDF format.
